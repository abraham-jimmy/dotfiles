# Claude

Claude Code client-specific config.

## Files

- `commands/`: symlinks to reusable shared commands in `~/.config/ai/commands`.
- `skills/`: symlinks to reusable shared skills in `~/.config/ai/skills`.

## Notes

- `~/.config/claude` is tracked so the shared symlink layout is consistent across machines.
- Shared AI source-of-truth content lives in `~/.config/ai`.
